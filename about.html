01 — My story

From writing code to designing what the code should feel like.

I started in computer science, which means I can read and write the code behind the screens I design. Somewhere along the way I realised I cared less about how the system worked and more about how it felt to the person using it — so I crossed over, and never looked back.

Five-plus years later I've shipped design across logistics, fintech and AI — at Porter, Juspay and Convin — building products used by millions across borders. The engineering roots never left: I prototype in code, argue in data, and think in systems.

Outside work you'll find me chasing light with a camera, on a badminton court, or shipping small ideas end-to-end with Claude — including a wallpaper app built from photos I clicked myself.
